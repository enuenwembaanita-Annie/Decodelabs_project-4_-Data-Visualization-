# Data Visualization - Sales Performance Dashboard

## Project Overview
This project creates visual representations of a sales dataset containing products such as monitors, phones, tablets, chairs, printers, and laptops to communicate insights clearly using Power BI.

## Key Requirements
- Create charts (bar, line)
- Choose appropriate visuals for data
- Highlight key insights

## Tools Used
- Power BI

## Dashboard Features
- KPI Cards (Total Revenue, Total Orders, 
  Average Order Value)
- Total Revenue by Product (Column chart)
- Orders by Order Status (Column chart)
- Orders by Payment Method (Bar chart)
- Orders by Referral Source (Bar chart)
- Revenue Trend by Year and Month (Line chart)
- Product and Year slicers

## Key Findings
- Total Revenue: $1,264,761.96
- Total Orders: 1,200
- Average Order Value: $1,053.97
- Chair generates the highest revenue
- Cancelled orders are highest at 250
- Instagram drives the most customers at 259
- June 2024 experienced the highest revenue 
  peak of $68K
